# go-dns
Unified DNS client for GOlang. See here for the documentation of [common provider interface](https://godoc.org/github.com/appscode/go-dns/provider).

### Supported DNS Providers
- [x] Google Cloud DNS
- [x] Cloudflare
- [x] DigitalOcean
- [x] Vultr
- [x] Linode
- [x] Azure
- [x] AWS Route53
